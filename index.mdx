## What SQLFluff does

SQLFluff parses your SQL into a segment tree, runs configurable rules against it, and can automatically fix most violations. It works seamlessly in local development, CI/CD pipelines, and pre-commit hooks.

<Steps>
  <Step title="Install SQLFluff">
    Install via pip — no additional dependencies required for most dialects.
    ```bash
    pip install sqlfluff
    ```
  </Step>
  <Step title="Lint your SQL">
    Point SQLFluff at a file or directory to see violations with line numbers and rule codes.
    ```bash
    sqlfluff lint query.sql --dialect postgres
    ```
  </Step>
  <Step title="Auto-fix violations">
    Let SQLFluff rewrite your SQL to comply with the configured rules.
    ```bash
    sqlfluff fix query.sql --dialect postgres
    ```
  </Step>
  <Step title="Add to CI/CD">
    Drop SQLFluff into your pre-commit config or GitHub Actions workflow to enforce standards automatically on every commit.
  </Step>
</Steps>
